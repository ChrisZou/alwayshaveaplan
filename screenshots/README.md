# Screenshots

Please add screenshots here:

1. **no-events.png** - Screenshot showing "你想干什么？" (What do you want to do?) prompt when there are no current events
2. **current-event.png** - Screenshot showing a current event with progress bar and remaining time

## How to take screenshots:

1. Run the app with `swift run`
2. For no-events view: Make sure you have no events in "日程安排" calendar
3. For current-event view: Add an event to "日程安排" calendar and unlock your Mac
4. Use macOS screenshot shortcuts:
   - Cmd+Shift+4: Select area to capture
   - Cmd+Shift+4, then Space: Capture specific window

Save the screenshots as PNG files in this directory with the names above.
